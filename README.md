# 📝 Smart Task Tracker

## 📌 About The Project
Smart Task Tracker is a simple web application for managing daily tasks.  
It helps users organize their work by adding, editing, deleting, and completing tasks easily.

This project was built to practice JavaScript and improve front-end development skills.

---

## ✨ Features
- ➕ Add new tasks  
- ✏️ Edit tasks  
- 🗑️ Delete tasks  
- ✅ Mark tasks as completed / not completed  
- 🌙 Dark / Light Mode  
- 💾 Save tasks using Local Storage  
- 📱 Simple and responsive design  

---

## ▶️ How To Use
1. Open the project in a browser  
2. Type a task in the input field  
3. Click "Add" or press Enter  
4. Use buttons to edit, delete, or complete tasks
